# 项目结构文档

## 1. 项目概览

这是一个基于 **Vue 2 + Vue Router + Vuex + Element UI** 的后台管理系统前端项目，整体风格与 **若依（RuoYi-Vue）** 保持一致。项目以 `src/main.js` 作为入口，通过路由、权限控制、全局组件注册和布局系统来组织页面与业务能力。

## 2. 顶层目录结构

```text
.
├── bin/                # 本地预览/启动相关脚本
├── build/              # 构建与打包辅助脚本
├── public/             # 静态资源与 HTML 资源
├── src/                # 业务源码
├── .env.development    # 开发环境变量
├── .env.staging        # 测试/预发布环境变量
├── .env.production     # 生产环境变量
├── babel.config.js     # Babel 配置
├── package.json        # 依赖、脚本与项目元信息
├── vue.config.js       # Vue CLI 构建配置
└── README.md           # 简要运行说明
```

## 3. `src/` 目录结构

```text
src/
├── api/                # 接口请求封装
├── assets/             # 图片、图标、样式等静态资源
├── components/         # 可复用通用组件
├── directive/          # 自定义指令
├── layout/             # 整体布局与页面框架
├── plugins/            # 全局插件封装
├── router/             # 路由配置
├── store/              # Vuex 状态管理
├── utils/              # 工具方法与通用能力
├── views/              # 页面视图
├── App.vue             # 根组件
├── main.js             # 程序入口
├── permission.js       # 路由权限控制
└── settings.js         # 全局配置
```

## 4. 各模块职责说明

### 4.1 `api/`
负责封装后端接口调用，按业务域拆分：
- `api/login.js`：登录相关接口
- `api/menu.js`：菜单与路由相关接口
- `api/system/*`：系统管理模块接口
- `api/monitor/*`：监控类接口
- `api/tool/gen.js`：代码生成相关接口

### 4.2 `assets/`
存放静态资源与公共样式：
- `assets/styles/`：全局样式、变量、布局样式
- `assets/icons/`：SVG 图标资源与加载配置
- `assets/images/`：登录页、头像、主题图等图片资源
- `assets/logo/`：项目 Logo

### 4.3 `components/`
通用组件库，供多个页面复用，例如：
- `Pagination`：分页组件
- `RightToolbar`：表格工具栏
- `Editor`：富文本编辑器
- `FileUpload` / `ImageUpload` / `ImagePreview`：上传与预览组件
- `DictTag` / `DictData`：字典标签与字典数据相关组件
- `Hamburger`、`Screenfull`、`ThemePicker` 等界面辅助组件

### 4.4 `directive/`
自定义指令集合，用于增强页面交互能力：
- `permission/`：权限控制指令
- `dialog/`：弹窗拖拽、宽高调整等指令
- `module/clipboard.js`：剪贴板相关指令

### 4.5 `layout/`
项目的整体页面框架：
- `layout/index.vue`：主布局容器
- `layout/components/`：布局内部组件，如侧边栏、顶栏、内容区
- `layout/mixin/ResizeHandler.js`：响应式布局处理

### 4.6 `plugins/`
全局插件与扩展能力封装，例如缓存、弹窗、下载、标签页等。

### 4.7 `router/`
路由系统入口，负责：
- 定义公共路由 `constantRoutes`
- 定义动态路由 `dynamicRoutes`
- 处理路由重复跳转报错
- 配合权限系统控制菜单与页面访问

### 4.8 `store/`
Vuex 状态管理：
- `modules/app.js`：布局与侧边栏状态
- `modules/user.js`：用户信息与登录状态
- `modules/permission.js`：权限与动态路由
- `modules/settings.js`：主题、布局等配置
- `modules/tagsView.js`：标签页导航
- `modules/dict.js`、`modules/lock.js`：字典与锁屏状态

### 4.9 `utils/`
通用工具方法与基础能力封装：
- `request.js`：请求封装
- `auth.js`：认证相关处理
- `ruoyi.js`：项目通用方法
- `validate.js`：校验方法
- `dynamicTitle.js`：页面标题处理
- `dict/`：字典转换与元数据处理
- `generator/`：代码生成器前端渲染工具

### 4.10 `views/`
页面级视图目录，按业务模块组织：
- `views/login.vue`：登录页
- `views/register.vue`：注册页
- `views/index.vue`：首页仪表盘
- `views/error/`：错误页
- `views/lock.vue`：锁屏页
- `views/dashboard/`：图表与首页组件
- `views/system/`、`views/monitor/`、`views/tool/` 等：后台业务模块页面

## 5. 核心启动流程

### 5.1 `src/main.js`
应用入口主要完成以下事情：
1. 引入 Vue、Element UI、全局样式与项目插件
2. 注册全局指令和全局组件
3. 挂载字典、下载、树结构处理等通用方法到 `Vue.prototype`
4. 初始化路由、状态管理和根组件
5. 创建 Vue 实例并挂载到 `#app`

### 5.2 `src/router/index.js`
路由分为两类：
- **公共路由**：登录、首页、错误页、锁屏页等
- **动态路由**：根据角色权限动态加载的业务页面

### 5.3 `src/layout/index.vue`
布局页负责统一页面骨架，包括：
- 侧边栏
- 顶部导航
- 标签页视图
- 主内容区
- 系统设置面板

### 5.4 `vue.config.js`
构建配置主要负责：
- 开发服务器端口与代理
- 生产构建输出目录
- 静态资源压缩
- SVG 图标处理
- 代码分包与缓存优化

## 6. 环境与构建文件

- `.env.development`：开发环境配置
- `.env.staging`：测试/预发布环境配置
- `.env.production`：生产环境配置
- `package.json`：脚本、依赖、项目元信息
- `babel.config.js`：Babel 编译配置
- `vue.config.js`：Vue CLI 打包配置

## 7. 常用启动命令

```bash
npm install
npm run dev
npm run build:stage
npm run build:prod
```
